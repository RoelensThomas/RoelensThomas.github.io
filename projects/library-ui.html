Sint-Lukas Arts Library

Relating the seemingly unrelated

For the academic art library of Sint-Lukas Brussels I designed a novel user interface that takes a new approach at searching and exploring the libaryâ€™s collection with the support of a redesigned information architecture. The new system not only allows users to search for a specified document, topic or maker but also enables them to explore the collection based on the connection between these three entities. This facilitates different types of information behaviour ranging from goal oriented searches to serendipitous discoveries.
